# Images Directory

Place your work photos in the following directories:

- `gallery/` - General portfolio/gallery images
- `services/` - Service-specific images (construction, property management, cleaning)

## Image Naming Convention

Suggested naming:
- `construction-01.jpg`, `construction-02.jpg`, etc.
- `property-management-01.jpg`, etc.
- `cleaning-01.jpg`, etc.
- `gallery-01.jpg`, `gallery-02.jpg`, etc.

## Image Requirements

- Recommended size: 1200x800px or larger
- Format: JPG or PNG
- Optimize images for web (compress before uploading)
