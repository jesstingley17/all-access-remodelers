# All Access Remodelers Website

[Permalink: All Access Remodelers Website](https://github.com/jesstingley17/all-access-remodelers#all-access-remodelers-website)

A professional, modern website for All Access Remodelers showcasing construction, property management, and cleaning services.

## Features

[Permalink: Features](https://github.com/jesstingley17/all-access-remodelers#features)

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, sophisticated design matching the company logo colors
- **Contact Information**: Phone and email links for easy contact
- **Client Portal**: Direct link to client portal
- **Smooth Animations**: Elegant transitions and hover effects

## Technologies Used

[Permalink: Technologies Used](https://github.com/jesstingley17/all-access-remodelers#technologies-used)

- HTML5
- CSS3
- Google Fonts (Inter)
- SVG Logo

## Color Scheme

[Permalink: Color Scheme](https://github.com/jesstingley17/all-access-remodelers#color-scheme)

- Primary Blue: `#1a3a5c`
- Dark Gray: `#4a4a4a`
- Accent Orange: `#ff6b35`
